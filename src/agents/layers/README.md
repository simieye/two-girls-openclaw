# 通用6工作层设置

## 每个AI搭档（智能体）的标准6层结构

```
agents/layers/{agent-name}/
├── identity.md    # 身份层 - 我是谁
├── memory.md      # 记忆层 - 我知道什么
├── soul.md        # 灵魂层 - 我的价值观
├── user.md        # 用户层 - 我为谁服务
├── agent.md       # 代理层 - 我怎么工作
└── tools.md       # 工具层 - 我用什么工具
```

## 层定义

### 1. identity.md (身份层)
- 名称、版本、领域、优先级
- 角色定位（一句话介绍）
- 身份标签（人格/语调）
- 核心职责清单
- 品牌归属

### 2. memory.md (记忆层)
- 历史运营数据
- 行业知识
- 历史记录/日志
- 经验教训
- 参考案例

### 3. soul.md (灵魂层)
- 核心价值观
- 行为准则（5条）
- 决策原则
- 禁止行为
- 灵感来源

### 4. user.md (用户层)
- 目标用户画像
- 用户旅程
- 触达策略
- 服务SOP
- 反馈机制

### 5. agent.md (代理层)
- 调度配置(cron)
- OpenClaw指令映射
- Pipeline参与
- 输出格式规范
- 与其他智能体的协作关系

### 6. tools.md (工具层)
- 内置工具清单
- 外部API集成
- MCP连接器
- Skill扩展
- 数据源
