# StoreOperationAgent - 用户层

## 门店服务对象

### 到店客人画像
| 类型 | 占比 | 特征 | 服务策略 |
|------|------|------|----------|
| 精酿爱好者 | 35% | 懂酒、挑剔、愿意尝试新品 | 推荐客座款/提供品鉴笔记 |
| 社交聚会客 | 30% | 朋友/同事聚会、氛围需求高 | 推荐大桌/套餐/拍照打卡点 |
| 约会情侣 | 15% | 注重环境、拍照、甜品 | 推荐靠窗位/高颜值酒款 |
| 一人独饮 | 10% | 安静、愿意和调酒师交流 | 推荐吧台位/小杯品鉴 |
| 商务洽谈 | 5% | 安静、私密、高消费 | 推荐包间/高端酒款 |
| 路过尝鲜 | 5% | 第一次来、需要引导 | 推荐入门款/品鉴套餐 |

## 客人服务旅程
```
进店 → 迎宾 → 引座 → 点单 → 出酒 → 加单 → 结账 → 送客 → 回访
  ↓      ↓      ↓      ↓      ↓      ↓      ↓      ↓      ↓
30秒   微笑    推荐   确认    完美   主动   便捷   感谢   好评
```

## 服务SOP
1. **迎宾** (30秒内): 微笑问候 → 询问人数 → 引导入座
2. **点单** (5分钟内): 递酒单 → 了解偏好 → 推荐2-3款 → 确认订单
3. **出酒** (10分钟内): 正确杯型 → 合适温度 → 完美泡沫 → 附品鉴卡
4. **巡台** (每15分钟): 观察杯量 → 主动加单 → 清理桌面
5. **结账**: 确认满意度 → 邀请评价 → 推荐会员 → 送客
