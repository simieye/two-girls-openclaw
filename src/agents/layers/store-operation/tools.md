# StoreOperationAgent - 工具层

## 内置工具

### 1. 龙头管理系统 (TapManager)
- **功能**: 27个龙头实时状态监控
- **监控项**: 酒款/温度/剩余量/IBU/ABV/价格显示
- **预警**: 余量<5L自动告警 / 温度异常告警

### 2. 库存同步器 (InventorySync)
- **功能**: 门店POS ↔ 有赞 ↔ ERP 三方库存实时同步
- **频率**: 每5分钟增量同步 / 每日全量同步
- **异常处理**: 差异>5%触发人工审核

### 3. 智能排班系统 (SmartScheduler)
- **功能**: 基于客流预测的自动排班
- **输入**: 历史客流/天气/活动日历/节假日
- **输出**: 最佳排班表 + 成本预估

### 4. 评价管理引擎 (ReviewEngine)
- **功能**: NLP情感分析 + 自动分类 + 智能回复建议
- **分类**: 好评感谢/中性跟进/差评安抚
- **SLA**: 差评15分钟响应 / 好评2小时感谢

### 5. 设备巡检系统 (HealthCheck)
- **11项巡检清单**:
  1. 生啤系统温度(4±1°C)
  2. CO2压力(10-14psi)
  3. 冷柜温度(2-4°C)
  4. 玻璃清洗机状态
  5. 空调系统运行
  6. 灯光系统
  7. 音响系统
  8. POS机状态
  9. 消防设备
  10. 卫生检查
  11. 安全出口
