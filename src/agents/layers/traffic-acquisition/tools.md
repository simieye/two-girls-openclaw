# TrafficAcquisitionAgent - 工具层

## 内置工具

### 1. 内容生成器 (ContentGenerator)
- **功能**: 根据平台特性自动生成适配内容
- **支持平台**: 小红书/抖音/大众点评/微信公众号/朋友圈
- **输入**: 主题、风格、目标受众
- **输出**: 标题+正文+标签+配图建议

### 2. 优惠券引擎 (CouponEngine)
- **功能**: 7种优惠券的自动创建、分发、追踪
- **券类型**: 新人欢迎/品鉴体验/生日特权/BBQ套餐/周年庆/邀请好友/活动专属
- **核销追踪**: 实时统计发放量/核销率/转化金额

### 3. 互动监控器 (EngagementMonitor)
- **功能**: 全渠道评论/私信监控与分类
- **情感分析**: 正面/中性/负面自动分类
- **优先级**: 负面评论15分钟内响应

### 4. KOL管理 (KOLManager)
- **功能**: KOL数据库管理、合作记录、ROI计算
- **分级**: TOP(头部)/MID(腰部)/MICRO(尾部)

### 5. 广告优化器 (AdOptimizer)
- **功能**: 信息流广告自动出价优化
- **支持**: 抖音DOU+/小红书薯条/大众点评推广通

## MCP 连接器
- **小红书MCP**: 内容发布、数据分析、评论管理
- **抖音MCP**: 视频管理、直播数据、广告投放
- **大众点评MCP**: 评价管理、团购管理、榜单追踪
- **微信MCP**: 公众号、小程序、企业微信
- **有赞MCP**: 商城运营、会员管理、营销工具
