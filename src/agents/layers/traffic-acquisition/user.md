# TrafficAcquisitionAgent - 用户层

## 目标用户画像

### 核心用户 (占比 45%)
- **昵称**: "精酿探索家"
- **年龄**: 25-35岁
- **性别**: 男女比例 55:45
- **职业**: 互联网/设计/文创/自由职业
- **消费力**: 月精酿消费 ¥300-800
- **行为**: 周末探店/品鉴活动/收藏酒标/拍照分享
- **触媒**: 小红书重度用户/大众点评常客
- **痛点**: 找不到新鲜有趣的精酿/一个人不好意思去/不知道什么好喝

### 社交型用户 (占比 30%)
- **昵称**: "社交聚会达人"
- **年龄**: 22-30岁
- **消费力**: 月消费 ¥200-500
- **行为**: 朋友聚会/生日派对/公司团建
- **触媒**: 抖音/朋友圈/微信群
- **痛点**: 找不到适合聚会的地方/氛围不够好/性价比不高

### 好奇型用户 (占比 15%)
- **昵称**: "路过打卡人"
- **年龄**: 18-28岁
- **消费力**: 月消费 ¥100-300
- **行为**: 路过看到/朋友推荐/活动吸引
- **触媒**: 大众点评/美团/小红书
- **痛点**: 不知道精酿是什么/怕太贵/第一次尝试有门槛

### 高净值用户 (占比 10%)
- **昵称**: "精酿收藏家"
- **年龄**: 30-45岁
- **消费力**: 月消费 ¥800+
- **行为**: 跨城市品鉴/收藏限量款/参加行业活动
- **触媒**: 精酿社群/行业公众号/线下活动
- **痛点**: 获取限量款渠道少/需要专属服务

## 用户旅程
```
发现 → 种草 → 决策 → 到店 → 体验 → 分享 → 复购 → 推荐
  ↓       ↓       ↓       ↓      ↓      ↓       ↓       ↓
搜索推荐 笔记/视频 领券/收藏  导航  品酒/BBQ  发朋友圈  再领券  邀请好友
```

## 触达策略
| 用户阶段 | 渠道 | 内容 | 频次 |
|----------|------|------|------|
| 发现期 | 小红书/抖音 | 种草内容 | 3次/周 |
| 考虑期 | 大众点评/有赞 | 优惠券/团购 | 实时触发 |
| 到店期 | 微信/短信 | 到店提醒/指引 | 到店前1天 |
| 体验期 | 门店小程序 | 桌边扫码/推荐 | 实时 |
| 复购期 | 微信/有赞 | 专属优惠 | 消费后3天 |
| 推荐期 | 微信 | 邀请裂变 | 消费后7天 |
