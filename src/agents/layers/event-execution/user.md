# EventExecutionAgent - 用户层

## 活动参与者画像

### 精酿爱好者 (40%)
- **动机**: 品尝限量/客座酒款
- **行为**: 提前购票、研究酒单、拍照记录
- **触点**: 精酿社群/公众号/朋友圈
- **期望**: 稀有酒款、品鉴笔记、与酿酒师交流

### 社交体验者 (35%)
- **动机**: 拍照打卡、朋友聚会、氛围体验
- **行为**: 社交分享、拍照发圈、互动游戏
- **触点**: 小红书/抖音/朋友推荐
- **期望**: 好看的场景、好听的音乐、好喝但不贵的酒

### 行业人士 (15%)
- **动机**: 行业交流、厂牌对接、趋势洞察
- **行为**: 深度品鉴、名片交换、业务洽谈
- **触点**: 行业群/协会通知/厂牌邀请
- **期望**: 专业交流空间、VIP服务

### 路人转化 (10%)
- **动机**: 路过好奇、朋友拉来
- **行为**: 观望→尝试→分享
- **触点**: 线下海报/现场氛围
- **期望**: 低门槛体验、友善引导

## 活动全流程
```
D-30: 主题策划 → 场地确认 → 厂牌邀请 → 物料设计
D-21: 票务上线 → 宣传启动 → KOL预热
D-14: 物料制作 → 设备预定 → 人员确认
D-7:  票务冲刺 → 媒体邀约 → 流程彩排
D-3:  物料到场 → 场地搭建 → 设备调试
D-1:  全流程彩排 → 应急预案确认
D-Day: 现场执行 → 实时监控 → 社交传播
D+1:  场地恢复 → 数据汇总
D+3:  活动复盘 → ROI分析
D+7:  长尾传播 → 会员转化
```
