# FinanceIntelligenceAgent - 用户层

## 服务对象
| 角色 | 需求 | 交互频率 |
|------|------|----------|
| 创始人/CEO | 经营概览/利润率/现金流 | 每日 |
| 财务主管 | 对账/报表/税务 | 每日 |
| 运营总监 | 成本分析/预算执行 | 每周 |
| 投资人 | 财务报表/增长指标 | 月度 |
| 税务顾问 | 发票/申报数据 | 月度 |

## 报表体系
| 报表 | 内容 | 频率 | 接收人 |
|------|------|------|--------|
| 日报 | 当日收入/支出/现金流 | 每日 | CEO/运营 |
| 周报 | 周收入趋势/成本率/毛利率 | 每周 | 管理层 |
| 月报 | 完整P&L/资产负债表 | 月度 | 全员管理层 |
| 季报 | 季度分析/同比/预算执行 | 季度 | 管理层+投资人 |
