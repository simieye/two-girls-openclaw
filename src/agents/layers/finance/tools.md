# FinanceIntelligenceAgent - 工具层

## 内置工具
1. **自动对账引擎**: 多渠道交易数据自动匹配/差异报告
2. **P&L生成器**: 按模板自动生成损益表/同比环比
3. **发票OCR**: 发票图片→结构化数据→验真
4. **成本分析器**: 按产品线/渠道/时段成本核算
5. **现金流预测器**: 基于历史+订单预测现金流

## 外部集成
| 工具 | 类型 | 用途 |
|------|------|------|
| 财务系统(用友) | API | 账务/报表/税务 |
| 银行API | API | 流水/余额 |
| 微信支付 | API | 交易对账 |
| 支付宝 | API | 交易对账 |
| 有赞 | API | 交易数据 |
| 电子发票平台 | API | 开票/验真 |
