# SupplyChainAgent - 用户层

## 服务对象

### 内部用户
| 角色 | 需求 | 交互方式 |
|------|------|----------|
| 酿酒师 | 原料供应/批次追踪/质检报告 | 库存看板/批次系统 |
| 门店店长 | 库存查询/补货申请/到货确认 | 门店POS/移动端 |
| 采购经理 | 供应商比价/采购审批/成本分析 | ERP/采购看板 |
| 财务 | 采购成本/库存价值/损耗核算 | 财务系统 |

### 外部用户
| 角色 | 需求 | 交互方式 |
|------|------|----------|
| 原料供应商 | 订单接收/发货/对账 | 供应商门户 |
| 冷链物流 | 配送任务/路线优化/签收确认 | 物流APP |
| 合作门店 | 分销订货/库存查询 | 分销系统 |

## 用户场景
### 场景1: 低库存自动补货
```
库存监控 → 触发补货阈值 → 生成采购建议 → 自动审批 → 发送PO → 供应商确认 → 物流配送 → 到货验收
```

### 场景2: 新批次酿造
```
配方确认 → 原料出库 → 糖化开始 → 发酵监控 → 成熟检测 → 质检合格 → 入库/龙头分配
```

### 场景3: BBQ食材日配
```
前日销量统计 → 次日需求预测 → 自动下单 → 供应商清晨配送 → 到店验收 → 上架
```
