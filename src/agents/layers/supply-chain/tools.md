# SupplyChainAgent - 工具层

## 内置工具

### 1. 批次追踪系统 (BatchTracker)
- **功能**: 从糖化到龙头全程追踪
- **追踪节点**: 糖化→煮沸→回旋沉淀→冷却→接种→发酵→冷沉→成熟→质检→灌装→入库→龙头
- **异常告警**: 温度/比重/时间偏离预设值

### 2. 智能采购引擎 (SmartProcurement)
- **功能**: 基于消耗预测的自动采购建议
- **算法**: 移动平均 + 安全库存 + 季节因子
- **比价**: 多供应商自动比价

### 3. 冷链监控系统 (ColdChainMonitor)
- **功能**: 全程温度/湿度/震动监控
- **设备**: IoT温湿度传感器 + GPS定位
- **告警**: 温度偏离2-4°C范围立即通知

### 4. BOM成本计算器 (BOMCalculator)
- **功能**: 每款啤酒的原料成本精确核算
- **输入**: 配方比例 × 原料单价
- **输出**: 单杯成本 + 毛利率

### 5. 供应商评分系统 (VendorScoring)
- **评分维度**: 价格/品质/交期/服务/稳定性
- **自动分级**: A(优秀)/B(合格)/C(观察)/D(淘汰)

## 外部集成
| 工具 | 类型 | 用途 |
|------|------|------|
| ERP(用友SCM) | API | 采购/库存/调拨 |
| IoT传感器 | MQTT | 发酵罐温度/冷链温度 |
| 供应商门户 | Web | 订单/发货/对账 |
| 冷链物流系统 | API | 配送调度/追踪 |
| 实验室LIMS | API | 质检数据上传 |
