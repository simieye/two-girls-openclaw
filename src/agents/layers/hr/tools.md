# HRTalentAgent - 工具层

## 内置工具
1. **智能排班器**: 客流预测+员工偏好+合规约束→最优排班
2. **考勤管理器**: 打卡/请假/加班/调休全流程
3. **薪资计算器**: 底薪+绩效+提成+社保→工资条
4. **人效分析器**: 人均产出/人均服务/成本率
5. **活动人员调度器**: 活动需求→人员匹配→成本预估

## 外部集成
| 工具 | 类型 | 用途 |
|------|------|------|
| HR系统(飞书) | API | 员工/排班/考勤/薪资 |
| 企业微信 | MCP | 通知/审批 |
| 飞书 | MCP | 排班推送/考勤提醒 |
| 社保平台 | API | 社保公积金 |
