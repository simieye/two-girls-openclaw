# HRTalentAgent - 用户层

## 服务对象
| 角色 | 需求 | 交互方式 |
|------|------|----------|
| 员工 | 排班查询/请假/换班/薪资 | 飞书/企业微信 |
| 店长 | 排班审核/人效看板/招聘需求 | 管理后台 |
| 管理层 | 人效分析/人工成本/组织健康度 | 月报 |
| 财务 | 薪资数据/社保公积金 | API对接 |

## 员工生命周期
```
招聘 → 入职 → 培训 → 上岗 → 考核 → 晋升 → 留任/离职
  ↓      ↓      ↓      ↓      ↓      ↓        ↓
JD发布  合同   产品知识  排班   月度   等级   交接
面试   社保   服务SOP  考勤   绩效   薪资   分析
Offer  设备   安全培训  人效   反馈   权限
```
