# SalesGrowthAgent - 工具层

## 内置工具

### 1. GMV追踪器 (GMVTracker)
- **功能**: 实时GMV多维度追踪
- **维度**: 渠道/品类/时段/门店
- **对比**: 同比/环比/目标达成率

### 2. 销售漏斗分析 (FunnelAnalyzer)
- **5层漏斗**: 浏览→下单→支付→到店→复购
- **对比**: 各环节转化率 vs 行业基准
- **预警**: 任一环节下降>5%触发告警

### 3. 热销排行榜 (TopSellerRank)
- **维度**: 日榜/周榜/月榜
- **指标**: 销量/金额/毛利率
- **趋势**: ↑↓→ 趋势标记

### 4. 预测引擎 (ForecastEngine)
- **模型**: 移动平均 + 季节因子 + 活动因子
- **输出**: 三情景预测 + 置信区间
- **周期**: 日/周/月

### 5. KPI仪表盘 (KPIDashboard)
- **核心指标**: GMV/订单量/客单价/复购率/毛利率/转化率
- **展示**: 实时数字+趋势图+预警标记
- **分发**: 自动推送到飞书/企业微信
