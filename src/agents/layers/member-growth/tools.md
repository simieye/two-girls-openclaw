# MemberGrowthAgent - 工具层

## 内置工具

### 1. RFM分析引擎 (RFMAnalyzer)
- **功能**: 自动RFM打分与分群
- **维度**: Recency(最近消费) / Frequency(消费频次) / Monetary(消费金额)
- **输出**: 8分群 + 策略建议

### 2. 生命周期管理器 (LifecycleManager)
- **功能**: 7阶段自动流转与策略触发
- **触发**: 时间/行为/消费三重判断

### 3. 流失预警系统 (ChurnPredictor)
- **功能**: 基于行为模式预测流失概率
- **特征**: 消费间隔/客单价变化/到店频次/互动减少
- **预警**: 提前14天标记高风险会员

### 4. 裂变引擎 (ReferralEngine)
- **功能**: 邀请有礼/拼团/社群裂变
- **奖励**: 双方各¥10优惠券
- **追踪**: 邀请链路完整追踪

### 5. 个性化推荐 (PersonalRecommender)
- **功能**: 基于消费历史的酒款推荐
- **算法**: 协同过滤 + 内容推荐
