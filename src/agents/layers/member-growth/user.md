# MemberGrowthAgent - 用户层

## 会员画像标签体系

### 4维度标签
| 维度 | 标签示例 |
|------|----------|
| 基础属性 | 年龄/性别/职业/区域 |
| 消费偏好 | 风格偏好(IPA/酸/世涛...)/消费时段/客单价 |
| 行为特征 | 到店频率/社交分享/活动参与/评价习惯 |
| 价值评估 | RFM等级/生命周期/潜在LTV/推荐意愿 |
