## 🤖 9大智能体详情

### 1. 📱 线上引流到店智能体 (`TrafficAcquisitionAgent`)
- **覆盖渠道**: 小红书 | 抖音 | 大众点评 | 有赞微商城 | 微信私域
- **核心能力**: 
  - 多平台内容自动生成(种草文案/短视频脚本/评价回复模板)
  - 活动发起与ROI预估(认知/转化/留存/活动四档战役)
  - 优惠券智能分发(新人礼/生日/欢乐时光/回归/推荐等7种)
  - 互动监控与数据汇总
- **KPI目标**: 月新增会员500人，线上引流到店率15%

### 2. 🏪 线下门店体验闭环智能体 (`StoreOperationAgent`)
- **基于**: 27龙头旗舰店真实配置
- **核心能力**:
  - 27龙头酒单管理与轮换(含颜色编码: IPA绿/世涛黑/酸啤粉)
  - 全渠道库存同步(有赞+门店+ERP)
  - 员工排班(日班/晚班/夜班+周末增员+活动特配)
  - 评价情感分析与自动回复
  - 11项设备健康巡检(CO2/温度/清洁/安全等)

### 3. 🔧 供应链体系智能体 (`SupplyChainAgent`)
- **基于**: 真实33款酒单 + 3款罐装 + BBQ原料清单
- **核心能力**:
  - 5批次酿造实时追踪(发酵→干投→熟成→装桶)
  - 完整原料BOM表(麦芽/酒花/酵母/辅料/包装)
  - BBQ供应链(牛肉/羊肉/海鲜/蔬果+供应商+新鲜度追踪)
  - 冷链物流(顺丰冷运/京东冷链+蓝色保温箱)
  - 低库存自动补货触发

### 4. 🎉 营销自动化与活动执行智能体 (`EventExecutionAgent`)
- **基于**: 第十届福建精酿啤酒年会真实数据
- **核心能力**:
  - 啤酒节全流程策划(D-30 → D-Day → D+7复盘)
  - 场地平面图设计(品酒区/主摊/品牌展区/舞台区)
  - 预算模板(¥15万投入/¥10.9万收入/品牌投资期逻辑)
  - 5套预定义Pipeline(引流闭环/活动全流程/日常运营/新品上市/自动补货)

### 5. 👥 会员增长智能体 (`MemberGrowthAgent`)
- **核心能力**:
  - RFM模型(8个细分群体: champions/loyal/atRisk/cannotLose等)
  - 7阶段生命周期策略(认知→获客→激活→留存→增收→推荐→召回)
  - 画像标签体系(偏好/行为/人口/社交 4维度)
  - 多渠道获客成本分析(加权CAC ¥8.2)

### 6. 📈 销售增长智能体 (`SalesGrowthAgent`) ⭐ 最高优先级
- **核心能力**:
  - GMV实时追踪(日/周/月报表)
  - 收入构成分解(生啤50% / BBQ25% / 罐装12% / 其他13%)
  - TOP10热销酒款排名(含趋势箭头)
  - 3个销售漏斗分析(线上到店/活动转会员/自然进店转忠诚)
  - 销售预测(保守/基准/乐观三情景)

### 7. 💰 财务智能体 (`FinanceIntelligenceAgent`)
- **数据源**: 有赞订单 ↔ POS ↔ 活动门票 ↔ ERP ↔ HR 全部互通
- **核心能力**: P&L损益表 | 多渠道对账 | 按产品线/渠道利润分析 | 现金流

### 8. 📅 HR智能体 (`HRTalentAgent`)
- **核心能力**: 活动人员调度(18人团队配置+成本预估) | 日常排班 | 考勤

### 9. 🚀 扩张智能体 (`ExpansionAgent`)
- **核心能力**: 新址评估 | 合作门店管理 | 即时零售(外卖/配送)

## 🚀 快速开始

```bash
cd clever-brew-agent-matrix
npm install
cp .env.example .env   # 填入各系统API密钥
```

### 运行模式

```bash
# 查看系统仪表板
npm run dashboard

# 启动定时任务调度器(持续运行)
npm run scheduler

# 单条指令 (支持自然语言!)
npm run command:sales        # "查看今日销售报表"
npm run command:marketing    # "发起小红书营销活动"  
npm run command:inventory    # "检查库存状态"
npm run command:reviews      # "处理最新评价"

# 执行预定义协作流程(Pipeline)
npm run pipeline:daily       # 每日自动化运营 (6步)
npm run pipeline:online      # 线上引流到店闭环 (7步)
npm run pipeline:event       # 啤酒节全流程 (10步)
npm run pipeline:product     # 新品上市全流程 (7步)

# 直接运行自定义指令
node src/index.js "查看本月GMV报告"
node src/index.js "marketing.campaign:launch:xiaohongshu"
node src/index.js --pipeline eventFullCycle
```

### OpenClaw 指令格式

```
# 结构化格式
sales:report:daily
marketing.campaign:launch:douyin
store.menu:update:tap_02
supply.brew:monitor:BREW-2025-042
member.analyze:rfm
finance.reconcile:yesterday

# 自然语言格式 (自动NLP解析)
"今天卖了多少"
"02出岫还剩多少"
"帮我发一批新人优惠券"
"准备下周六的排班"
"分析一下会员流失情况"
```
