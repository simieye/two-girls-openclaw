## 7大模块 20+ 智能体

### 模块01: 供应链与采购 (3个)
| 智能体 | 状态 | 功能 |
|--------|------|------|
| 📦 原料监控智能体 | ✅ 已上线 | 实时监控库存，触发低位预警 |
| 📋 智能采购规划体 | ✅ 已上线 | 基于销售趋势和季节因素自动生成采购计划 |
| 💰 多供应商比价体 | 🔶 灰度中 | 比对报价、交期、品质，输出采购决策 |

## 定时任务

| 任务 | Cron | 说明 |
|------|------|------|
| supply-monitor | */5 * * * * | 每5分钟检查原料库存 |
| inventory-sync | */15 * * * * | 每15分钟同步全链路库存 |
| store-patrol | 0 */2 * * * | 每2小时自动巡店 |
| marketing-report | 0 6 * * 1 | 每周一早6点营销报告 |
| finance-reconcile | 0 2 * * * | 每日凌晨2点财务对账 |
| hr-schedule | 0 0 * * 1 | 每周一排班 |
| erp-sync | 0 */1 * * * | 每1小时ERP同步 |
| member-churn-check | 0 9 * * * | 每天早9点流失检查 |
| daily-report | 0 8 * * * | 每天早8点绩效报表 |

## 数据互通链路

```
有赞微商城 ──→ 订单数据 ──→ 财务系统 (对账)
     │                        │
     ├──→ 商品库存 ──→ ERP供应链 (采购)
     │                        │
     ├──→ 销售数据 ──→ HR系统 (排班)
     │
有赞门店小程序 ──→ 门店订单 ──→ 财务系统
     │
     └──→ 会员数据 ──→ 营销CRM (推送)
```

## 启动方式

```bash
# 安装依赖
npm install

# 复制环境配置
cp .env.example .env
# 编辑 .env 填入各系统API密钥

# 启动
npm start

# 开发模式 (热重载)
npm run dev

# 单独启动智能体
npm run agent:supply
npm run agent:inventory
npm run agent:store
npm run agent:marketing
npm run agent:finance
npm run agent:hr
npm run agent:expansion
```
